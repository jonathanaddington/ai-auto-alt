== Description ==

Make your website more accessible to visually impaired users with the AI Auto Alt Text plugin for WordPress. Powered by advanced artificial intelligence, this innovative plugin automatically generates descriptive alt text for images as they're uploaded, ensuring that your content is more inclusive and SEO-friendly.

**Features:**

- **Automatic Alt Text Generation**: Leverage OpenAI's latest models for accurate and contextually relevant alt descriptions.
- **Support for Multiple Image Types**: Works with JPG, PNG, GIF, and more.
- **EXIF Data Analysis**: Utilizes metadata for enhanced alt text creation.
- **Customizable Prompts**: Tailor the AI response to suit your site's needs.
- **Advanced Settings**: Control the AI's creativity and specificity with adjustable parameters.

**Benefits:**

- **Improved Accessibility**: Assists visually impaired users with screen reader-friendly descriptions.
- **SEO Boost**: Enhances image searchability and overall site SEO.
- **Time-saving**: Automates the creation of alt text for images.
- **Enhanced User Experience**: Provides users with informative image descriptions.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ai-auto-alt-text` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the plugin settings page to enter your OpenAI API key and configure the AI settings.
4. Begin uploading images to your media library and alt text will be automatically generated.

== Frequently Asked Questions ==

= Does this plugin support all image types? =

Yes, it supports most common image types including JPG, PNG, GIF, and more.

= Is this plugin only for visually impaired users? =

While the plugin greatly aids visually impaired users, it also enhances SEO and user experience for all users.

= Can I customize the AI-generated alt text? =

Yes, you can customize the prompts used by the AI through the plugin settings page to suit the context of your website.
